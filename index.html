Overview

The GEL Test Rig is designed to provide simple and clear feedback for maintainers of webpages that implement GEL visual and interaction patterns. It aims to define all GEL patterns using the testable BDD-style specification—Given, When, Then—in order to encourage test-driven design. In addition, the test rig provides an easy to use tool that can interrogate your web page and provide red/green feedback to quickly reveal precisely which patterns your implementation match, and which ones your implementation does not match.

Set Up

The Test Rig requires a "fixture page" that will serve as the test subject, and which will be examined and compared to the GEL patterns to check for differences.

If you have created an implementation of a GEL pattern, for example if you have a CSS and HTML page that demonstrates the GEL Typography pattern, you can create a fixture page by simply publishing a "sampler page" that contains all the examples of typography.
